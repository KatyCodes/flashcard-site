JS & jQuery Flashcards

JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always. Assignment Operators change the value of the variable on the left side of the operator. Comparison Operators do not change any values, but returns a boolean.

Variables

containers to store informations. They allow for a way to label data with a descriptive name.

Naming conventions

variable and function names written as camelCase.

Functions

block of code that performs an action and returns a result; optionally takes arguments.

Methods

a property containing a function definition

Arguments

the value in the parenthesis of a method. They provide a bit more info to methods to help them know what they're supposed to do.

Parameters

a variable that's assigned to the argument, and receives the argument.

Return

the response to the method's call.

Chaining methods

calling a method directly on the return value of another method.

Strings

a javascript data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

represents one of two values: true or false

Undefined

a variable has been declared but has not yet been assigned a value.

NaN

stands for "not a number" but is in fact a number.

Escape

function encodes a string

Alert

opens a dialog box

Comments

anything following // on a line in JavaScript is ignored by the computer. Comments help you and other programmers understand your code more effectively.

jQuery

a concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing.

Attributes

additional information about an HTML element.
